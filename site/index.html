Slenderman

— a poorly drawn typeface —

Slenderman is a free, open-source font. I made it because of computers. It comes in one style and is best suited for larger sizes.

I was drinking a lot when I made it so there’s a good chance it’s missing glyphs that you need. It’s on GitHub so put in an issue or better yet fork it, make the update(s), and submit a pull request.

Download ( otf, ttf, woff, svg, and eot )
